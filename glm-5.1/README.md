# glm-5.1 Track

This directory contains the `glm-5.1` benchmark track.

- Put `glm-5.1` artifacts directly under this directory.
- Store each benchmark record in `runs/*.json` using `python benchmark_ledger.py create-run --model glm-5.1 ...`.

This track starts empty and is ready for the first WebGPU / WASM benchmark submission.
